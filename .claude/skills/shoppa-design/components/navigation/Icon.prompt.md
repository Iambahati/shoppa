Self-contained Heroicons (outline, 1.5 stroke) used throughout the marketplace UI. `solid` switches to the filled trust glyphs.

```jsx
<Icon name="shield" size={20} />
<Icon name="search" />
<Icon name="check-badge" solid size={16} />   {/* the verified mark */}
```

Names: home, search, box, layers, users, store, package, flag, shield, cpu, message-sq, bell, user, check, x, chevron-r, chevron-d, bars, qr, plus, arrow-right. Solid: check-badge, shield-check. Colour follows `currentColor`.
