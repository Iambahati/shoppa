Inline status message with a leading icon and soft tinted surface. Use for flash messages and form-level feedback.

```jsx
<Alert type="success">Your password has been reset.</Alert>
<Alert type="warning" title="Awaiting verification">This device is queued for inspection.</Alert>
<Alert type="error">An IMEI must be unique across active listings.</Alert>
```

Types: `info | success | warning | error`. Pass `title` for a bold lead line.
